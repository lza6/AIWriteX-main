## 📥 下载指南

### 方式一：直接下载压缩包

1. **选择版本**: 点击上方对应版本的下载链接
2. **解压文件**: 使用 WinRAR/7-Zip 解压下载的压缩包
3. **安装依赖**: 运行 `setup.bat` (Windows) 或手动安装依赖

### 方式二：Git 克隆 (推荐开发者)

```bash
# 克隆最新代码
git clone https://github.com/iniwap/AIWriteX.git
cd AIWriteX

# 或切换到特定版本
git checkout v23.0.0
```

### 方式三：使用安装包 (Windows 用户)

1. 访问项目 Release 页面下载 `.exe` 安装包
2. 双击安装包按照提示完成安装
3. 在软件界面配置 API Key 即可开始使用

## 🔧 升级指南

### 从旧版本升级到 V23.0

1. **备份数据**: 备份 `data/` 和 `secrets/` 目录
2. **下载新版**: 下载 V23.0 压缩包或使用 git pull
3. **更新依赖**: 运行 `pip install -r requirements.txt --upgrade`
4. **迁移配置**: 对照 `.env.example` 更新配置文件
5. **测试运行**: 运行 `python main.py` 测试是否正常

### 注意事项

- ⚠️ **V17+ 版本**: 需要配置向量数据库 (可选)
- ⚠️ **V19+ 版本**: 建议配置 Redis 用于缓存
- ⚠️ **V21+ 版本**: 推荐使用 SSD 硬盘以获得最佳性能
- ⚠️ **V22+ 版本**: 需要更多内存 (建议 16GB+)
