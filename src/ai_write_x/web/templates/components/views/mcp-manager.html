服务列表

💡 使用说明

  • Fetch: 网页内容抓取，支持HTML转Markdown。需要安装 uvx (点击上方"安装"按钮自动安装)
  • Playwright: 浏览器自动化，支持网页导航、截图、表单操作。需要安装 Node.js
  • Chrome DevTools: Chrome浏览器调试工具，支持性能分析、网络监控。需要安装 Node.js
  • Memory: 知识图谱内存服务，支持实体关系存储。需要安装 Node.js
  • Verge News: The Verge科技新闻服务，获取实时新闻和搜索。需要安装 Node.js
  • Pulse CN: 中国互联网热门内容，支持微博、知乎、B站等18个平台。需要安装 Node.js

💡 提示：点击"安装"按钮可自动安装依赖，或按提示手动安装。启用服务后需要点击"启动"按钮来运行服务
