# AIWriteX 微信公众号AI写作助手 | 多智能体爆文生成神器 🚀

## 📖 项目概述

AIWriteX 是一款**新一代智能内容创作平台**，专为微信公众号内容创作打造，融合了 AI 技术与传统写作流程，实现从选题到发布的全自动化。

### 🌟 核心价值

- **效率提升**：一键全自动热点选题 + 实时搜索借鉴 + AI创意生成 + 精美排版 + 微信公众号自动发布
- **质量保证**：CrewAI 多智能体协作，从研究员到作家到审核员，层层把关
- **时效性强**：全网热点雷达，提前2-6小时抓取黑马话题
- **去AI味**：深度对抗检测引擎，让文章更具人性化
- **素材管理**：私有化内容资产管理系统，构建专属素材库

### 软件模式（下载发布版 exe）

1. **安装运行**：双击下载的 `.exe` 安装程序，按照提示完成安装。
2. **配置信息**：在软件界面中配置 `API Key` 和公众号信息。
3. **开始创作**：点击`开始执行`按钮。

### 源码模式（开发者/GitHub 下载）

1. **环境准备**：确保系统中已安装 Python 3.10+。
2. **一键配置**：双击运行项目根目录下的 `setup.bat`。
   - 该脚本会自动创建虚拟环境 (`.venv`)。
   - 自动安装 `requirements.txt` 中的所有依赖。
   - 自动从 `.env.example` 复制生成 `.env` 配置文件。
3. **填入密钥**：打开生成的 `.env` 文件，填入您的 API 密钥及其他配置。
4. **启动项目**：双击运行 `启动.bat`。

## 🎯 核心功能详解

### 1. 全网热点雷达 🌩️

- **聚合多平台热点**：微博/抖音/小红书/热搜数据
- **黑马挖掘**：热度0但12小时内预计暴涨的话题
- **趋势预测**：精准预判未来12小时流量走势
- **深度分析**：生成爆点提纲 + 完整写作指导

### 2. 多智能体协作 🤖

- **研究员**：搜索最新资讯，收集相关素材
- **作家**：基于素材生成原创内容
- **审核员**：检查内容质量，确保符合平台规范
- **设计师**：进行精美排版，优化视觉效果

### 3. 去AI味/AI对抗检测 🛡️

- **动态风格拟态**：提取参考文章的"用词DNA"，实现风格复刻
- **结构粉碎**：打散工整的列表和段落，破坏AI的"完美秩序"
- **情感色彩增强**：注入强烈主观色彩的表达与反问句
- **评分反馈进化**：根据对抗评分优化后续流程

### 4. 素材文库 📚

- **微信文库**：一键采集任意公众号文章，图片/内容永久本地存储
- **智能选题**：AI深度分析本地库，推荐高价值选题
- **内容挖掘**：语义级搜索，找到真正相关的深度素材
- **AI融合洗稿**：选中多篇爆文，AI自动去重、提炼、重组

### 5. 维度化创意变换系统 🌈

- **多风格转换**：同一素材，瞬间变身多种风格
- **适用场景**：
  - 内容创作者：一篇素材 → 公众号/小红书/抖音多版本
  - 营销/教育：针对不同人群/视角调整表达
  - 脑洞爱好者：李白穿越现代、魔幻现实主义书信体

### 6. 智能排版 🎨

- **自动适配**：微信/小红书/抖音展示风格
- **模板系统**：多种精美模板，支持分类选择
- **视觉优化**：Glassmorphism + 暗黑模式 + 滚动微动画

### 7. 自动发布 📤

- **多账号管理**：支持配置多个微信公众号
- **定时任务**：设置发布时间，自动发布
- **批量生成**：一次生成多篇文章

## 📊 技术原理

### 1. 多智能体架构

AIWriteX 使用 CrewAI 框架构建多智能体系统，通过角色分工和协作流程，实现高质量内容创作：

- **Agent 1: 研究员**：负责信息收集和分析
- **Agent 2: 作家**：负责内容创作
- **Agent 3: 审核员**：负责内容质量检查
- **Agent 4: 设计师**：负责排版和视觉优化

### 2. 热点预测算法

基于物理算法 + AI研判，实现热点预测：

- **数据采集**：实时抓取多平台热点数据
- **趋势分析**：分析话题热度变化趋势
- **潜力评估**：评估话题爆发潜力
- **预测模型**：基于历史数据和AI分析，预测未来热度

### 3. 去AI味技术

通过多种技术手段，减少AI生成内容的机械感：

- **风格模仿**：提取人类写作风格特征
- **结构调整**：打破AI的固定结构模式
- **情感注入**：增加主观情感表达
- **自然语言处理**：优化语言表达，使其更自然

### 4. 模板引擎

自适应模板引擎，支持多种风格和平台：

- **模板分类**：按话题类型分类，如科技数码、财经投资、教育学习等
- **响应式设计**：适配不同设备和平台
- **动态生成**：根据内容自动调整模板布局

## 🔧 配置说明

### config.yaml 配置项

| 配置项 | 说明 |
|-------|------|
| **platforms** | 设置各平台热搜话题随机选取权重，控制选用优先级 |
| **wechat** | 支持配置多个微信公众号（自动发布时，必填`appid`、`appsecret`、`author`） |
| **wechat.credentials** | `call_sendall`、`sendall`、`tag_id`用来控制群发情况（仅适用于认证号） |
| **api** | 支持多种大模型平台，必填`api_key` |
| **api.api_type** | 支持多个大模型平台，修改 `api_type` 切换平台，如OpenRouter |
| **api.OpenRouter.model_index** | 修改 `model_index` 选择平台内具体模型 |
| **api.OpenRouter.model** | 支持多种模型（如openrouter/deepseek/deepseek-chat-v3-0324:free） |
| **api.OpenRouter.key_index** | 修改 `key_index` 切换账号（充分利用免费额度，付费账号无需多个） |
| **api.OpenRouter.api_key** | 支持多个 OpenRouter `api_key` |
| **img_api** | 图片生成模型，用于公众号封面图 |
| **img_api.api_type** | `ali`（需要填写`api_key`）或`picsum`（随机图片） |
| **img_api.picsum** | 随机图片生成方式，降低生成图片消耗 |
| **use_template** | 是否使用内置模板 ，不使用则AI根据要求直接生成文章HTML |
| **template** | 指定模板文件名（如 `template1`），为空或不存在时随机选择 |
| **template_category** | 模板分类，精确匹配话题类型（如健康养生），需分类下存在指定模板 |
| **use_compress** | 是否压缩模板上传，降低 token 消耗 |
| **aiforge_search_max_results** | AIForge 最大返回搜索结果条数，控制搜索广度 |
| **aiforge_search_min_results** | AIForge 最小返回搜索结果条数，越大内容越丰富，但失败率越高 |
| **min_article_len** | 生成文章最小字数（默认 1000） |
| **max_article_len** | 生成文章最大字数（默认 2000） |
| **auto_publish** | 控制自动发布，勾选（true）自动发布，不勾选(false)需手动发布 |
| **article_format** | 生成文章格式（HTML、Markdown、txt）,非HTML时，只生成文章，不用模板 |
| **format_publish** | 当文章格式为Markdown、txt时（微信不支持，直接发布混乱），格式化发布 |

## 🔍 常见问题与解决方案

### 1. 微信公众号发布问题

- **IP白名单问题**：微信 API 需将当前 IP 添加至后台白名单
  - 解决方案：使用云服务器转发请求，或通过阿里云函数计算（免费，需注册阿里云）代理微信 API 请求

- **权限回收**：自 2025 年 7 月起，个人主体账号、未认证企业账号及不支持认证的账号将失去发布草稿权限
  - 解决方案：尽快完成公众号认证，或使用认证账号发布

- **文章显示问题**：发布的文章（未认证公众号）不会显示在公众号文章列表，但关注者会收到通知
  - 解决方案：使用认证账号，调用群发接口

### 2. AI模型相关问题

- **API Key 配置**：确保正确配置 API Key，不同模型平台的配置方式不同
- **模型选择**：根据内容类型选择合适的模型，如创意内容可选择更具创意的模型
- **生成质量**：如生成内容质量不佳，可调整 prompt 或模型参数

### 3. 模板相关问题

- **模板适配**：部分模板可能在微信公众号中显示异常
  - 解决方案：使用已适配的模板，或自行调整模板

- **暗色模式**：微信暗色模式下模板显示效果可能不佳
  - 解决方案：目前生成更倾向于适用微信浅色模式
